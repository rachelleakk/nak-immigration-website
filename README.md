# NAK Immigration Consultancy Website

A modern, responsive business website developed for NAK Immigration Consultancy.

## Live Demo
[View Website](https://rachelleakk.github.io/nak-immigration-website/)

##  Built With
- HTML5
- Tailwind CSS (CDN)
- JavaScript
- Stripe Payment Links Integration
- Google Sheets API (Testimonials)

##  Features
- Fully responsive design
- Animated sections and counters
- FAQ accordion
- Stripe product purchase integration
- Dynamic testimonials from Google Sheets
- Clean, modern UI
